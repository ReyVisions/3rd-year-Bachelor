On reprend le programme du tp 0 dans ce TP.

Pour verifier les resultats des parcours,
on teste par exemple dans le main() pour un arbre donne:
en input, ecrire: 0 ; 1 ; 00 ; 01 ; 10 ; 11 donne l'arbre suivant:

            1
          /   \
         2     3
        / \   / \
       4   5 6   7

il faudra verifier si le parcours prefixe donne bien:
1,2,4,5,3,6,7
le parcours infixe:
4,2,5,1,6,3,7
et le parcours postfixe:
4,5,2,6,7,3,1
